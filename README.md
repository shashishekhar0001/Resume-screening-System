# 🚀 Smart Resume Screening System

AI-powered resume screening system that ranks candidates based on job descriptions using NLP.

## 🔥 Features
- Multi-resume ranking
- Skill extraction
- Matching score with progress bars
- FastAPI backend
- Dockerized deployment

## 🛠️ Tech Stack
- Python, FastAPI
- NLP (TF-IDF, Cosine Similarity)
- HTML, CSS, JS
- Docker
- AWS EC2

```bash
uvicorn app:app --reload
